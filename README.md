# 💳 Banking Application (Java + MySQL)

## 📌 Overview
A console-based banking system built using Java and JDBC with MySQL integration.  
It supports basic banking operations like account creation, deposit, withdrawal, and transaction tracking.

## 🚀 Features
- Create Account
- Deposit & Withdraw Money
- Balance Inquiry
- Transaction History

## 🛠 Tech Stack
- Java
- JDBC
- MySQL

## ⚙️ Setup Instructions

1. Run `schema.sql` in MySQL
2. Update DB credentials in `BankApp.java`
3. Compile and run:

```bash
javac BankApp.java
java BankApp
